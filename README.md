# BIT-MiniCC
BIT Mini C Compiler is a C compiler framework in Java for teaching.

# Building & Running
## Requirements
* JDK 1.8 or higher
* Eclipse Mars

## Building & Running
1. Import the project into Eclipse
2. Set the input source file
3. run as Java applications from class BitMiniCC
